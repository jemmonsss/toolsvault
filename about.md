<div class="about-page">
    <h1>About ToolsVault</h1>
    <div class="content">
        <p>ToolsVault is a curated collection of developer tools, designed to help you discover, compare, and access the best resources for your workflow.</p>
        <p>Each tool in the vault is tracked in a simple markdown file with consistent metadata, making it easy to add new tools without touching any code.</p>
        <h2>How to Add a Tool</h2>
        <p>1. Copy <code>_tools/example.md</code> to <code>_tools/your-tool.md</code><br>
        2. Edit the frontmatter and description<br>
        3. Commit and push</p>
        <p>Contributions welcome! Open a pull request on <a href="https://github.com/yourusername/toolsvault">GitHub</a>.</p>
