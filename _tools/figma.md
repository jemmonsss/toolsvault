## What is Figma?

Figma is a browser-based design and prototyping tool that enables real-time collaboration. Designers, developers, and stakeholders can work together on the same file simultaneously.

## Key Features

- Vector-based design and auto-layout
- Interactive prototyping and animations
- Design systems and component libraries
- Real-time multiplayer collaboration
- Developer handoff with CSS/JSON snippets

## Getting Started

Sign up for free at [figma.com](https://www.figma.com) and start designing in your browser.
