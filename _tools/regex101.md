## What is Regex101?

Regex101 is a comprehensive, web-based regular expression tester. It supports PCRE, JavaScript, Python, Go, and Java flavors, and provides a detailed breakdown of your pattern.

## Key Features

- Multiple regex flavors (PCRE, JavaScript, Python, Go, Java)
- Real-time match highlighting
- Detailed explanation of each token
- Quick reference and cheat sheet
- Saved patterns with shareable links

## Getting Started

Visit [regex101.com](https://regex101.com), paste your regex and test string, and explore the matches and explanation panels.
