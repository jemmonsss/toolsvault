## What is Prettier?

Prettier is an opinionated code formatter with support for JavaScript, TypeScript, CSS, HTML, JSON, Markdown, and more. It parses your code and reprints it with its own rules, ensuring consistent formatting across entire projects.

## Key Features

- Supports JS, TS, CSS, HTML, GraphQL, Markdown, YAML
- Editor integrations for VS Code, Vim, Emacs, and more
- CLI and programmatic API
- Configurable with `.prettierrc`
- Prettier plugin ecosystem

## Getting Started

```bash
npm install --save-dev prettier
npx prettier --write .
```
