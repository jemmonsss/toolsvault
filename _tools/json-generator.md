## What is JSON Generator?

JSON Generator is a tool that generates mock JSON data based on templates and rules. It's useful for quickly prototyping APIs, writing tests, and populating databases with realistic fixtures.

## Key Features

- Template-based JSON generation
- List and range rules
- Random data from word lists
- Embed pseudo-random bytes and dates
- Export generated data

## Getting Started

Visit [json-generator.com](https://json-generator.com), define your template on the left, and copy the generated JSON on the right.
